部屬Blog遇到問題

在部屬Hexo發現的問題之一,原本想用.gitignore來限制自己不想顯示的檔案,
但後來發現已經送交至版本庫的檔案，事後再來設定 .gitignore 檔案是沒用的!!!
利用

git rm --cache <file>

再把它給commit上去就OK囉~

安裝Hexo必須具備

  • Node.js
  • Git

補充網址:

  • 忽略不想上傳的文件
  • 一開始就設定.gitignore
